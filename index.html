What's New in Digital Luxe

  • Fix
    Components render correctly on canvas

    Components now appear exactly as they look in the catalog — no more card wrappers or title bars dragging onto the canvas.

    Buttons, Cards, Forms, Navigation, Dashboards, Typography, Modals, Tables, and Blocks — all with smart default sizing.

    Safety dialogs

    Clearing the canvas or navigating away now asks if you want to save first — no more accidental work loss.
